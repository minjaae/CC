# CC : CarCinema

## 프로젝트 소개
서울, 경기권 지역의 자동차 극장 정보를 한 번에 볼 수 있는 안드로이드 어플리케이션 (구글 플레이스토어 출시)
 
## 프로젝트 개발 환경
Java, Firebase, Android Studio              
                 
## 페이지 상세
* 홈         
* 자동차 극장 리스트    
* 자동차 극장 정보          
* 주변 맛집 리스트
